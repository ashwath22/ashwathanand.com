Dash button hack

Duration: NOvember 2016
My role: creative technologist
Tech stack: python, node.js, nightmare.js

Project details

Part of a marketing campaign pitch, I hacked an amazon dash button to order the same item from ebay for a cheaper price.



Prototype of the hacked dash button to show what it will feel like in real time.


Process

I wrote a small python script to find out when the MAC address of a particular dash button was detected. Once the signal is detected, a javascript app built on nightmare.js runs a pre defined ebay page to complete the ordering process.
